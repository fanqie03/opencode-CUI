# OpenCode-CUI Demo 实施计划

**版本**: v1.0
**日期**: 2026-03-04
**项目周期**: 8 周
**关联文档**: `需求澄清文档.md`, `架构调整说明.md`

---

## 一、项目目标

开发一个本地联调 Demo，验证 OpenCode → PCAgent → AI-Gateway → Skill Server → Web UI 的完整技术链路，为后续 IM 平台集成奠定基础。

### 核心交付物
1. ✅ 独立运行的 PCAgent 程序
2. ✅ AI-Gateway 和 Skill Server 后端服务
3. ✅ React Web UI 前端应用
4. ✅ AK/SK 管理后台
5. ✅ 完整的部署文档和用户手册

---

## 二、项目里程碑

| 阶段 | 时间 | 目标 | 交付物 |
|------|------|------|--------|
| **Phase 1** | Week 1-2 | 核心链路验证 | 端到端消息传输 |
| **Phase 2** | Week 3-5 | 功能完善 | 流式渲染、Tool Use、历史恢复 |
| **Phase 3** | Week 6-7 | AK/SK 管理后台 | 用户自助管理系统 |
| **Phase 4** | Week 8 | 测试和优化 | 完整的 Demo 系统 |

### 3.1 PCAgent 改造（3 天）

#### 任务清单
- [ ] **Task 1.1**: 创建独立的 Node.js 项目结构
  - 初始化 `package.json`
  - 配置 TypeScript

- [ ] **Task 1.2**: 实现 OpenCode 事件订阅
  - 集成 `@opencode-ai/sdk`

  - 实现心跳保活机制

- [ ] **Task 1.5**: 实现配置和启动
