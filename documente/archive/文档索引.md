## 🎯 核心概念速查

### 透明中继模式
- **定义**：服务端不解析 AI 工具输出，原样透传
- **位置**：AI-Gateway 和 Skill Server
- **优势**：协议独立、低延迟、简化服务端逻辑
- **详见**：`docs/架构设计详解.md` 第 5.1 节

### 双服务架构
- **AI-Gateway**：连接管理、AK/SK 认证、透传中继（端口 8081）
- **Skill Server**：会话持久化、业务逻辑、IM 集成（端口 8082）
- **详见**：`docs/架构设计详解.md` 第 2 节

### 客户端协议适配
- **位置**：`skill-miniapp/src/protocol/`
- **职责**：解析 OpenCode 事件、拼接流式消息、转换 Tool Use
- **详见**：`skill-miniapp/src/protocol/AGENTS.md`

### Redis Pub/Sub 协调
- **通道**：`agent:{agentId}`, `session:{sessionId}`
- **用途**：多实例路由、消息广播
- **详见**：`docs/技术方案文档.md` 第 2.3 节
